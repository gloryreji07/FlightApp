# **App Name**: FlightTrackr

## Core Features:

- Flight Search: Implement a search form to find flights by origin/destination or flight number, utilizing the provided API.
- Flight Display: Show flight search results in a clear and responsive card layout.
- Flight Details: Allow users to view detailed flight information including route, times, and status.
- Watchlist Management: Enable users to add and remove flights from a watchlist, with persistence using local storage.

## Style Guidelines:

- Primary color: Calm blue (#5DADE2) to represent the sky and trustworthiness.
- Background color: Light grey (#F5F7FA), offering a clean and modern feel.
- Accent color: Soft orange (#F39C12) to highlight key actions and information, like call-to-action buttons.
- Headline font: 'Poppins', a geometric sans-serif with a contemporary feel.
- Body font: 'PT Sans', a humanist sans-serif that offers a balance of modernity and readability.
- Use a set of simple, clear icons from 'react-icons' to represent flight-related actions and information, such as departure, arrival, and aircraft type.
- Design a responsive layout that adapts to both mobile and desktop screens, using Tailwind CSS grid and flexbox.
- Implement smooth transitions and loading skeletons to improve the user experience during API calls and page transitions.
